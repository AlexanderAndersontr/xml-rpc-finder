# xml-rpc-finder
XML-RPC Bulucu

WordPress sitelerinde sıklıkla güvenlik açığı oluşturan ve yöneticiler tarafından devre dışı bırakılması unutulan xml-rpc.php dosyasının varlığını tespit eden tarama aracıdır. -f parametresi ile site listesini içeren dosya belirtilir, araç listedeki tüm siteleri otomatik olarak tarar ve xml-rpc.php dosyasının aktif olup olmadığını raporlar.
